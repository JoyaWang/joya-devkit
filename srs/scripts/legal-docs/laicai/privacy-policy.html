重要声明：本应用在您首次启动时会以弹窗形式向您展示本隐私政策，在您阅读并同意本隐私政策之前，我们不会收集或使用您的任何个人信息，包括但不限于设备标识符（Android 端的 IMEI、OAID、设备序列号；iOS 端的 IDFA、IDFV 等）、设备 MAC 地址、软件安装列表、通讯录、短信等信息。只有当您点击"同意"后，相关功能所必需的信息收集才会启动。

引言

"来财"App（以下简称"本应用"）由无尽探索（西安）科技有限公司（以下简称"我们"）开发和运营。我们深知个人信息对您的重要性，并会尽全力保护您的个人信息安全。我们致力于维持您对我们的信任，恪守以下原则保护您的个人信息：权责一致原则、目的明确原则、选择同意原则、最少够用原则、确保安全原则、主体参与原则、公开透明原则等。

本隐私政策适用于您通过"来财"App（Android 版和 iOS 版）使用我们的产品和服务。

一、我们收集的信息

为向您提供服务，我们需要收集和使用以下个人信息：

1.1 账号注册与登录

  • 手机号码：用于账号注册、登录验证和安全保护。我们通过发送短信验证码的方式验证您的身份。

1.2 个人资料信息

  • 昵称、头像：用于展示您的个人形象，方便其他用户识别您。
  • 实名认证信息：当您选择进行实名认证时，我们会收集您的真实姓名及身份证号码，用于身份核验，提升交易信任度。

1.3 位置信息

  • 地理位置：本应用基于地理位置提供本地化服务（如附近的需求和供给信息、市场定位），需要获取您的位置信息。我们使用百度地图 SDK 提供地图和定位服务。您可以在系统设置中随时关闭位置权限。

1.4 交易相关信息

  • 发布信息：您发布的需求或供给内容，包括标题、描述、分类、价格、图片等。
  • 订单信息：交易订单的详情，包括订单状态、交易金额、配送方式、服务地址等。
  • 人情分记录：您的人情分账户余额、收支记录，用于平台内的积分交易。

1.5 通讯信息

  • 聊天记录：您与其他用户之间的消息内容，用于提供即时通讯服务。我们不会将您的私聊内容用于任何其他目的。

1.6 设备与日志信息

为了保障服务安全、进行风险控制、优化用户体验以及推送消息通知，我们可能在您同意本隐私政策后收集以下设备相关信息：

  • 基础设备信息：设备型号、设备品牌、操作系统版本（如 Android 版本号）、屏幕分辨率等，用于保障服务兼容性和优化用户体验。
  • 设备标识信息：Android 平台：OAID（匿名设备标识符）；iOS 平台：IDFA（广告标识符）、IDFV（供应商标识符）。用于消息推送、安全风控和统计分析。我们仅在您同意本隐私政策后才会收集上述标识信息。
  • 设备 MAC 地址：百度地图 SDK 和百度定位 SDK 在提供定位和地图服务时，可能会读取设备的 MAC 地址，用于辅助定位精度和网络环境判断。该信息仅在您同意隐私政策并使用地图/定位相关功能时才会被收集。
  • 软件安装列表：个推（GeTui）消息推送 SDK 在建立推送通道时，可能会收集设备上的部分已安装应用信息，用于智能推送通道选择和推送成功率优化。该信息仅在您同意隐私政策后才会被收集，且不会用于任何广告投放或用户画像目的。
  • 网络信息：网络类型（WiFi/移动数据）、WiFi 连接状态、IP 地址，用于保障网络通信和服务安全。
  • 日志信息：使用服务时的操作日志、崩溃日志，用于故障排查和服务改进。

1.7 其他信息

  • 相机和相册权限：当您上传图片（如发布信息的图片、头像、反馈截图）时，需要访问您的相机或相册。
  • 推送通知：我们使用个推 (GeTui) 推送服务向您发送订单状态变更、新消息等通知。
特别说明：本应用不会收集您的通讯录、短信、通话记录等与核心服务无关的敏感个人信息。我们严格遵守最少够用原则，仅收集实现产品功能所必需的信息。

二、我们如何使用收集的信息

我们严格按照法律法规的规定，将收集的信息用于以下目的：

  1. 提供核心服务：包括用户注册、身份验证、信息发布、订单交易、即时通讯、地图定位等基本功能。
  2. 安全保障：用于身份验证、风控审核、防范欺诈、保护账号安全。
  3. 服务优化：分析使用数据以改进产品功能和用户体验。
  4. 消息通知：向您发送与服务相关的通知，如订单状态更新、系统通知等。
  5. 客户服务：处理您的反馈、投诉和建议。

三、信息的存储与保护

3.1 存储位置与期限

您的个人信息存储在中华人民共和国境内的服务器上（腾讯云）。我们仅在为您提供服务所必需的期间内保留您的个人信息。在您注销账号后，我们将在合理的时间内删除或匿名化处理您的个人信息，法律法规另有规定的除外。

3.2 安全措施

我们采取以下措施保护您的信息安全：

  • 使用 HTTPS 加密传输数据
  • 对敏感信息（如密码、身份证号）进行加密存储
  • 实施访问权限控制，仅授权人员可访问用户数据
  • 定期进行安全审计和漏洞检测

五、您的权利

根据相关法律法规，您享有以下权利：

  1. 访问权：您可以在"个人中心"查看和访问您的个人信息。
  2. 更正权：您可以修改您的昵称、头像等个人资料信息。
  3. 删除权：在特定情况下，您可以请求我们删除您的个人信息。
  4. 撤回同意：您可以在设备系统设置中关闭位置、相机、相册、通知等权限。
  5. 注销账号：您可以通过"个人中心"或联系客服申请注销账号。注销后，我们将停止为您提供服务，并依法处理您的个人信息。

六、未成年人保护

我们非常重视未成年人的个人信息保护。本应用的服务面向成年人，如果您是未满 18 周岁的未成年人，请在监护人的指导下使用本应用，并请监护人帮助您阅读本隐私政策。若我们发现在未事先获得监护人同意的情况下收集了未成年人的个人信息，我们将尽快删除相关数据。

七、隐私政策的更新

我们可能会不时更新本隐私政策。当政策发生重大变更时，我们会通过应用内通知等方式告知您。建议您定期查阅本政策以了解最新的隐私保护信息。
